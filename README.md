Uber Ride Analysis Dashboard
An interactive Excel Data Analytics Dashboard designed to visualize and analyze ride-hailing data for the year 2024. This project transforms raw ride records into actionable insights regarding customer behavior, booking trends, and revenue performance.

📊 Project Overview
This dashboard provides a comprehensive view of Uber ride operations, focusing on key performance indicators (KPIs) and spatial-temporal trends. It allows stakeholders to filter data by specific dates, months, or quarters to observe fluctuations in demand and service quality.

Key Metrics (KPIs)
Total Customers: 150,000

Total Revenue Value: 5,184,618

Average Booking Value: 508.30

🚀 Key Features
Dynamic Slicers: Filter the entire report by Month (Jan–Dec) or specific Calendar Dates for granular analysis.

Booking Status Breakdown: A bar chart displaying the volume of Completed, Cancelled (by Customer/Driver), Incomplete, and "No Driver Found" rides.

Geospatial Insights:

Top Pickup Locations: Identification of high-demand areas like Khandsa and Barakhamba Road.

Top Drop Locations: Analysis of popular destinations such as Ashram and Basai Dhankot.

Temporal Trends: A line graph showing monthly booking fluctuations throughout 2024.

Payment Analysis: A pie chart detailing preferred payment methods (UPI, Uber Wallet, Cash, Credit/Debit cards).

Customer Satisfaction: A distribution chart of customer ratings, highlighting service quality trends.

🛠️ Tools Used
Microsoft Excel: Primary tool for data cleaning, modeling, and visualization.

Pivot Tables & Pivot Charts: Used for dynamic data aggregation.

Slicers & Timelines: Implemented for interactive user filtering.

Data Modeling: Organized raw data to calculate Average Booking Value (ABV) and customer counts.

📈 Insights Derived
High Completion Rate: The majority of bookings are successfully completed, though there is a notable volume of cancellations by drivers and customers that could be optimized.

Peak Demand: Ride volume peaks significantly in July, suggesting seasonal demand or specific marketing campaigns.

Payment Preferences: UPI and Uber Wallet represent a significant portion of the transaction volume, indicating a high adoption of digital payments.

Rating Distribution: While many users provide high ratings (4.5–5.0), there is a significant "null" or unrated segment that presents an opportunity for improving feedback loops.

📂 How to Use
Download the .xlsx file from this repository.

Open the file in Microsoft Excel 2016 or later.

Use the Slicers on the left-hand side to interact with the charts and update the KPIs in real-time.
